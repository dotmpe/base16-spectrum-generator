# Base16 Spectrum Generator

A Python script for generating `.png` files that showcase the colours in a Base16 theme.

## Usage 

General Usage:

```Bash
python main.py <theme_file> <output_file>
```

Explicit example:

```Bash
python main.py decaf.yaml decaf.png
```
